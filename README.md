# Airbnb Clone ‚Äì MERN Stack

A full-stack Airbnb clone built using the MERN (MongoDB, Express, React, Node.js) stack. This project allows users to book, host, and manage listings with a seamless UI inspired by Airbnb.

## Features

- ‚úÖ **User Authentication** ‚Äì Secure login/signup with JWT
- ‚úÖ **Listings Management** ‚Äì Hosts can create, update, and delete listings
- ‚úÖ **Booking System** ‚Äì Users can book accommodations with availability checks
- ‚úÖ **Image Uploads** ‚Äì Hosts can upload multiple images for their listings
- ‚úÖ **Search & Filters** ‚Äì Find properties based on location, price, and amenities
- ‚úÖ **Protected Routes** ‚Äì Users and hosts have different access levels
